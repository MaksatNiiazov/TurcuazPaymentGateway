# Сообщение для MKassa

Здравствуйте.

Просим подключить API QR интеграцию MKassa.

Callback URL для получения статусов QR-транзакций:

```text
https://<ДОМЕН-БУДЕТ-ПРЕДОСТАВЛЕН-ПОЗЖЕ>/api/v1/webhooks/mkassa
```

Сервер будет доступен по HTTPS с действующим SSL-сертификатом на порту 443.

Callback endpoint принимает `POST` запросы с JSON-данными о статусе транзакции и возвращает:

```text
HTTP 200 OK
```

Просим предоставить/подтвердить:

- `cashier_api_key` / `api_key` для запросов к MKassa API;
- идентификаторы торговых точек `branch`;
- идентификаторы кассиров `cashier`;
- возможность получения callback-уведомлений на указанный URL;
- возможность использовать `metadata` при создании QR-транзакций;
- подтверждение, что код фактуры из Tiger можно передавать в `metadata.invoice_number`;
- возможность использовать `is_long_living` для динамических QR-транзакций, если требуется увеличение времени ожидания оплаты.

После готовности домена дополнительно отправим финальный Callback URL.
